Видеорежимы

384x304 (ZX-Guide #2)
Видеорежим 384x304 для Pentagon, цветной, стандартное ограничение на два цвета в знакоместе. Позволяет использовать для отображения всю область экрана, включая бордюр.
Дополнительный графический режим - 512x192 (Online #79,#80)
Видеорежим 512x192 для Pentagon, монохромный. Для отображения используется штатная область экрана, без использования бордюра.
Дополнительный графический режим - 512x192 (Deja Vu #06)
Другая версия предыдущей статьи, содержит некоторые отличия.
Расширенный экран (512x192) (ZX Power #2)
Ещё одна версия описания схемы видеорежима 512x192.
128 цветов на Spectrum'е (Deja Vu #0A)
Увеличение количества отображаемых цветов до 128 за счёт использования бита Flash (Flash color). Описание процесса выполнения доработки для ZS Scorpion.
Bright + Flash (ZX Power #3)
Увеличение количества отображаемых цветов за счёт использования бита Flash. Несовместима с Flash color! Описание процесса выполнения доработки для Pentagon, Leningrad, Profi разных версий.
