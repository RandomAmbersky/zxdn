        Дополнения по подключению
           дисководов на  3.5'

     Решив поставить себе дисковод на 3.5
дюйма, я был несколько  удивлен  тем, что
он выбирался у меня как драйв [С], вместо
того, что-бы выбираться как [A] (на  моем
дисководе  "Mitsumi"  нет джамперов). При
попытке сделать  переключатель, как напи-
сано в литературе ( ZX-Ревю, Adventurer и
еще где-то ) ничего не получилось. "Треш-
ка"  упорно не хотела выбираться ( честно
говоря,если повесить один из проводов,от-
вечающих за выбор дороги на "воздух" , то
"трешка"  перестает выбираться  вообще ).
Немного  повозившись,  я  все  же  сделал
сhange  дисководов.  Для  этого  пришлось
припаять 12-ый контакт шлейфа на 14-ый, а
14-ый  контакт,  через  переключатель  на
12-ый.

     +---------+--------------- 10
     |         |
     |  +----+-|--------------- 12
     |  |    | |   +----------- 14
     |  |    | |   | 14
     |  |    | |   |
     \-------\     |
      |      |     |
      |      |     |
     10 14 12
