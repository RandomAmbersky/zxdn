Общие идеи и цели проекта

Цель проекта ZXDN - собрать все материалы по творчеству на/для/с помощью компьютера ZX Spectrum в едином сборнике, отсортировав их по тематике и снабдив аннотациями. Основной источник материалов - электронная ZX-пресса.

При помещении в коллекцию материалы приводятся в единый формат. Для материалов без иллюстраций - текстовые файлы в кодировке ANSI-1251 (Windows) с переводом строки по CR/LF. Для материалов с иллюстрациями - html-файлы с минимальным форматированием. Все имена файлов в формате 8.3, используются только цифры и латинские буквы.

Для html-файлов применяются только следующие возможности:

  • Текст в <pre>..</pre>
  • Изображения всегда в формате gif, разрешение по возможности не превышает стандартное для ZX
  • Тег <img> для иллюстраций (они помещаются в одноимённую с файлом папку, с именами pic1.gif ... picN.gif)
  • При необходимости замена в тексте символов < > на &lt; &gt; и " на &quot;


    • Все эти меры принимаются для упрощения поиска требуемого текста стандартными средствами (поисковыми системами, поиском фрагмента текста в файлах средствами ОС). Также они предусматриваются для возможности создания впоследствии мини-браузера для просмотра коллекции на реальном ZX (с дискет, жёсткого диска, CD).

    Оригинальная орфография и форматирование сохраняются, если это возможно. Также по возможности сохраняются оригинальные названия. Более конкретно содержание статьи описывается в аннотации. Если оригинального названия нет - придумывается короткое и говорящее, описывающее тему статьи. Декоративное оформление текста псевдографикой убирается или заменяется на аналогичное, но ASCII-набором. Источник материала полностью указывается в начале каждого текста, авторство - в том месте, где и в оригинальном тексте, если указано не было - то в начале текста. Авторство материалов принадлежит их уважаемым авторам.

    Материалы в данный момент выбираются по принципу: 'что попалось под руку и было несложно извлечь из источника'. Если вам понадобится какой-то материал по тематике коллекции, отсутствующий в ней - не поленитесь его найти, извлечь из источника и прислать для включения в состав сборника.
